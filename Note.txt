--> It is a programming pattern that resolves around objects or entities.

## Popular Programming approaches

    1. Procedure Oriented Programming  -> C 
    2. Object Oriented Programming -> Java, python, C#

    3. Code Reusability & Easy Code Modification
    4. Better Software Maintenance

    1. Class

    # According to Encapsulation
        --> Data : Data members must be kept private.
        --> Methods :- Methods only can access these data members and these methods can be made public.
